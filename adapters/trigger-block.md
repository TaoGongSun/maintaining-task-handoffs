## Long-task handoffs

Use judgment to mark work as long. Do not activate this workflow for short or unrelated tasks. Once active, use the `maintaining-task-handoffs` skill and run `handoff checkpoint` after material changes and before compaction. Run `handoff complete` before finishing. If the skill cannot load, read `$HOME/.agents/skills/maintaining-task-handoffs/SKILL.md`. Keep detailed state in `<repo>/.ai/HANDOFF.md`; the final chat contains only its path and one next action.
