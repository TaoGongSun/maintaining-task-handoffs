## Long-task handoffs

On long-task start or wrap-up, use the `maintaining-task-handoffs` skill. If the agent cannot load skills automatically, read `$HOME/.agents/skills/maintaining-task-handoffs/SKILL.md` and follow the same contract. After updating the handoff, the chat response must keep only the handoff path and a single next action on the last line; detailed state stays in the handoff file.
